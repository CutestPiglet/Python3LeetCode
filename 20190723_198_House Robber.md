#### Problem
No. 198: https://leetcode.com/problems/house-robber/

#### Code
```python
class Solution:
    def rob(self, nums: List[int]) -> int:
        """
        f(0) = nums[0]
        f(1) = max(num[0], num[1])
        f(k) = max(f(k - 2) + nums[k], f(k - 1))
        """
        
        last, result = 0, 0
        for num in nums:
            last, result = result, max(last + num, result)
            
        return result
```
